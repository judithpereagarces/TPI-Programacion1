# Gestión de Datos de Países en Python

## Descripción

Este proyecto fue desarrollado para el Trabajo Práctico Integrador de Programación 1.

La aplicación permite gestionar información de países mediante un menú interactivo en consola. Los datos son almacenados en un archivo CSV y se trabajan utilizando listas, diccionarios y funciones en Python.

## Funcionalidades

* Mostrar países.
* Agregar países.
* Actualizar población y superficie.
* Buscar países por nombre.
* Filtrar por continente.
* Filtrar por rango de población.
* Filtrar por rango de superficie.
* Ordenar por nombre.
* Ordenar por población.
* Ordenar por superficie.
* Mostrar estadísticas generales.
* Guardar cambios automáticamente en el archivo CSV.

## Tecnologías utilizadas

* Python 3
* Archivo CSV
* Visual Studio Code

1. Descargar el proyecto.
2. Abrir la carpeta en Visual Studio Code.
3. Ejecutar el archivo `main.py`.
4. Utilizar el menú para acceder a las diferentes funcionalidades.
